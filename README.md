# 🔐 CAPTCHA Authentication System - Python

A Python program that simulates a **mathematical CAPTCHA system** to verify that the user is human by solving random calculation operations.

---

## 📌 Project Description

A CAPTCHA is a security method used to distinguish humans from automated programs.

This project creates a simple calculation-based CAPTCHA:

🧮 A random mathematical operation is displayed  
✍️ The user enters the answer  
✅ The program checks the response  
🔐 Authentication result is displayed  

---

# 🚀 Features

✨ The program can:

- 🎲 Generate random mathematical operations
- ➕ Handle addition operations
- ➖ Handle subtraction operations
- ✖️ Handle multiplication operations
- 📦 Store operations in a first array (`T1`)
- 📊 Store results in a second array (`T2`)
- 🔀 Select a random operation
- ✅ Validate user answers
